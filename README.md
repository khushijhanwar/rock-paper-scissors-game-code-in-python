# Rock-Paper-Scissors Game 🎮✊✋✌️

This is a simple Rock-Paper-Scissors game implemented in both Python script (`.py`) and Jupyter Notebook (`.ipynb`) formats. The game allows a user to input a choice and plays against the computer, which selects a choice at random.

---

## 🛠 Features

- Text-based user interface.
- Input validation for rock, paper, or scissors.
- Clear win/loss/tie logic using conditional statements.
- Available in both script and notebook format.

---

## 📦 Requirements

You only need Python's built-in libraries:

- Python ≥ 3.6
- `random` (standard library)

To run the notebook version:

- Jupyter Notebook
- IPython kernel
